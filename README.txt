Death Calculator
Welcome to the Death Calculator! This is a simple Python program that estimates how long you may live based on your current age.

Getting Started

To calculate your remaining years, weeks, and months, follow these steps:
1. Run the program.
2. When prompted, enter your current age.
3. The program will calculate the remaining years, weeks, and months based on an assumed average life span of 90 years.
4. It will display the calculated results.

Prerequisites
Python 3.x

Usage

1.Open a terminal or command prompt.
2.Navigate to the directory where the code file is located.
3.Run the following command to execute the program: python deathCalculator.py

Example

Enter your current age: 30
You have 21600 years, 1560 weeks, and 360 months left

Please note that these calculations are based on the assumption of an average life span of 90 years.

Feel free to use and modify this Death Calculator according to your needs.
